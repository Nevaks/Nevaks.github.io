My Works

  • TensorFlow and Faster R-CNN Application as Walking Aid for The Blind with Limited Object Identification

    Created for the university thesis, an app that utilizes machine learning and smartphone's camera to help visually impaired people walk in public places. This application is able to know and estimate the distance of objects such as traffic lights, traffic signs, and several types of trees.
    Technologies used: Python, TensorFlow, and Faster R-CNN.

  • Confluence of Death

    Confluence of death is a 3D Side-scrolling action platformer game created using the Unreal Engine. In this game, players are required to survive from enemy attacks as long as possible. The longer the player can survive, the higher the score he gets.
    Technology used: Unreal Engine 4.

  • Juggling Bytes

    Juggling Bytes is a visual novel created using Ren'Py software (based on Python language). This game takes a story based on the experiences of students who study informatics. How difficult was it when you first adapted from high school to university.
    Technologies used: Python, Ren'py, and Photoshop.
